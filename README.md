# Amazon Clone

This project was built with React JS, Node JS, Stripe JS and Firebase.

## Features

The features include:

* 📝 Full E-Commerce application.
* 📡 Realtime database in Firebase.
* 📦 Full responsive
* 💬 Stripe payment method
* 📡 Hosted on Firebase.
